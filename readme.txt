== Description ==

As what its name says, it delete all your tags, or tags of zero posts.<br />
Be careful when using it.<br />
Feel free to visit <a href="http://www.s101.net">S101</a> to report bugs.

== Installation ==


1. Extract delete-all-tags.zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Delete all tags" In "Posts" Menu
4. Follow the on-screen instructions
